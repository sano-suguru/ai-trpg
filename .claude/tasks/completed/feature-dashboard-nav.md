# タスク: ヘッダーにマイページリンク追加

## 概要

ヘッダーナビゲーションにマイページへのリンクを追加する。

## 背景

- `/dashboard` ページが作成されたが、そこへの導線がない
- ログインユーザーがマイページにアクセスする手段が必要

## 前提条件

- `feature-dashboard-page.md` が完了していること

## 要件

### 機能要件

- [x] ヘッダーに「マイページ」リンクを追加
  - [x] ログイン時のみ表示
  - [x] アクティブ時はハイライト（既存スタイル踏襲）

## 設計

### 影響範囲

| パッケージ | ファイル | 変更内容 |
| ---------- | -------- | -------- |
| `web` | `src/components/layout/Header.tsx` | リンク追加 |

### ヘッダー変更

```
Before:
[AI-TRPG] [キャラクター] [ダンジョン] ... [user@email] [ログアウト]

After:
[AI-TRPG] [キャラクター] [ダンジョン] [マイページ] ... [user@email] [ログアウト]
                                        ↑ ログイン時のみ
```

## 実装手順
